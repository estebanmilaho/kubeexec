# Contributing

Thanks for considering a contribution!

## Issues
- Please include reproduction steps, expected vs actual behavior, and your OS + kubectl version.
- If relevant, include the command you ran and any error output.

## Pull requests
- Keep changes focused and add tests where practical.
- Run `go test ./...` before opening a PR.
- Use `gofmt` on changed Go files.

## Development
```bash
go test ./...
```
